# Données personnelles et formulaire (RGPD)

Ce site peut collecter via le formulaire de contact : **nom**, **adresse e-mail**, éventuellement **téléphone**, **sujet** et **message**.

## Finalités

- Répondre à la demande de contact.
- Assurer un suivi administratif lié à cette demande.

Pas de prospection commerciale ni de newsletter à partir de ces champs **sans consentement distinct et explicite**.

## Base légale

Intérêt légitime pour répondre à une demande entrante / mesure précontractuelle ou contractuelle selon le cas — à affiner dans vos mentions légales avec votre conseil si besoin.

## Durée de conservation

À définir dans votre politique de confidentialité (ex. durée nécessaire au traitement de la demande + limitation dans le temps pour les archives).

## Droits des personnes

Les personnes concernées peuvent exercer leurs droits (accès, rectification, effacement, limitation, opposition, etc.) via les coordonnées que vous publierez dans les mentions légales.

## Sous-traitance et transferts

- **Hébergeur** du site et du serveur mail : identifier dans les mentions légales.
- **SMTP** : le fournisseur mail peut traiter les données en transit — choisir un fournisseur conforme à vos obligations.

## Sécurité

Mesures techniques côté application décrites dans [SECURITE.md](SECURITE.md) (validation, limitation de débit, en-têtes HTTP).

## À compléter sur le site

La page « Contact & informations légales » doit contenir les informations définitives : responsable du traitement, coordonnées, finalités détaillées, durées, destinataires, droits. Ce fichier est une base opérationnelle pour l’équipe projet, pas un document juridique substitut.
